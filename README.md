# Cancer Cell Toxicity Prediction Pipeline

## 1. Project Overview
This project implements a multi-stage machine learning pipeline to quantify and predict the cellular toxicity of chemical compounds on tumor cells. This pipeline was used to analyze microscopy images from https://idr.openmicroscopy.org/webclient/?show=screen-1251.

## 2. Methodology 

### Phase 1: CNN Autoencoder 
*   **Choice:** An unsupervised Autoencoder was selected to avoid the need for manual toxicity labeling of every image.
*   **Mechanism:** The model is trained exclusively on "healthy" cell images to learn their baseline features. The model calculates the **Reconstruction Error** for the test images, which represents any biological damage inflicted by the drug treatments.
*   **Quantification:** Continuous error values are binned into a scale of **0-9**, providing a discrete "Toxicity Score" for supervised learning in the FNN model.

### Phase 2: FNN 
*   **Choice:** A Feed-forward Neural Network (FNN) was chosen to map molecular descriptors to biological outcomes.
*   **Mechanism:** **SMILES** notations were used to determine structural features of each drug compound, which were used as input and uses the CNN’s binned toxicity score as its target label.
*   **Value:** This enables the prediction of a new compound's toxicity purely from its chemical formula, potentially bypassing the need for physical lab experiments.
